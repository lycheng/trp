# trp

A simple TCP server written in Rust.

## Quick Start

```bash
cargo run
```

The server will start listening on `127.0.0.1:8080`.

## Usage

Connect to the server using any TCP client:

```bash
telnet 127.0.0.1 8080
```

## Build

```bash
cargo build
```
